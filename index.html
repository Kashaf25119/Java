Some Codes in JAVA

Butterfly Pattern

This Java program prints a symmetrical star pattern by combining an upper and lower section. The first set of nested loops creates the expanding upper part, while the second set generates the contracting lower part, using spaces and stars to form a clean mirrored diamond-like shape.

Bubble Sort

This Java program takes five numbers from the user, displays them, and then sorts them in ascending order using the Bubble Sort algorithm. It compares and swaps adjacent elements until the entire array is sorted, and finally prints the sorted list.
