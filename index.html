Some Codes in JAVA

Butterfly Pattern

This Java program prints a butterfly star pattern. The first set of nested loops creates the expanding upper part, while the second set generates the contracting lower part, using spaces and stars.

Bubble Sort

This Java program takes five numbers from the user and sorts them in ascending order using the Bubble Sort algorithm. It compares and swaps adjacent elements until the entire array is sorted, and finally prints the sorted list.

Palindrome

This Java program uses a recursive function to reverse a string and check whether it is a palindrome. It compares the original and reversed strings (case-insensitive) and displays whether the input is a palindrome or not.

Unique Sub Strings Generator

This Java program generates all unique subsequences of a given string using recursion and a HashSet to avoid duplicates. It explores all include/exclude possibilities for each character and prints each distinct subsequence.
